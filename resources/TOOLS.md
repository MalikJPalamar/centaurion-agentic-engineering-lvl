# Tools and Platforms by Level

## Level 1-2: IDE and Autocomplete
- [GitHub Copilot](https://github.com/features/copilot) - Most widely adopted
- [Cursor](https://cursor.sh) - Codebase-aware chat and editing
- [Kiro](https://kiro.dev) - Spec-driven development by AWS
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) - CLI-first terminal agent

## Level 3: Context Engineering
- CLAUDE.md, .cursorrules, AGENTS.md conventions
- [Spec Kit](https://github.com/github/spec-kit) - GitHub spec-driven dev toolkit

## Level 5: MCP and Skills
- [Composio](https://composio.dev) - 1000+ third-party tool integrations
- [DeepWiki MCP](https://deepwiki.com) - Documentation access for any OSS repo
- [Playwright MCP](https://github.com/anthropics/anthropic-quickstarts) - Browser automation

## Level 6: Harness Engineering
- [Greptile](https://greptile.com) - AI code review
- [CodeRabbit](https://coderabbit.ai) - Context-aware PR reviews
- [Lighthouse CI](https://github.com/GoogleChrome/lighthouse-ci) - Performance scoring

## Level 7: Background Agents
- [Agent Orchestrator](https://github.com/ComposioHQ/agent-orchestrator) - Fleet management
- [sweteam](https://sweteam.dev) - CLI orchestrator
- [Open SWE](https://github.com/langchain-ai/open-swe) - Enterprise pattern

## Level 9: Autonomous Deployment
- [Vercel](https://vercel.com) / [Netlify](https://netlify.com) - Git push to production
- [GitHub Actions](https://github.com/features/actions) - CI/CD workflows

## Level 10: Simulation
- [MiroFish / OASIS](https://github.com/) - Multi-agent world simulation
- [Synthetic Users](https://syntheticusers.com) - AI-generated user research

## Level 11: Physical-Digital Bridge
- [Home Assistant](https://home-assistant.io) - 3000+ device integrations
- [OctoPrint](https://octoprint.org) - 3D print management
- [ESPHome](https://esphome.io) - No-code IoT firmware
