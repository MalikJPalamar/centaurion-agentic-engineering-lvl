# Level 4: Compounding Engineering

## The Loop
```
Plan -> Delegate -> Assess -> Codify -> Update CLAUDE.md -> Next session is better
```

## The Codify Journal
After every significant agent interaction, ask three questions:
- **What worked:** What the agent got right
- **What broke:** What the agent got wrong
- **Root cause:** Missing context? Wrong tool? Ambiguous instruction?
- **Codified:** What you added to CLAUDE.md to prevent recurrence

## Anti-Patterns
- **Codify everything** Too many rules is as good as none
- **Codify the symptom, not the cause** Prefer root cause rules
- **Never prune** Review and remove stale rules monthly

## Level Up Checklist
- [ ] You have a codify log that you update after agent sessions
- [ ] Your CLAUDE.md has been refined at least 5 times
- [ ] You can point to a specific codified rule that prevented a repeat mistake
- [ ] You have pruned at least one stale rule
